## Current Product Decision

The dashboard should be a guided weekly brief first and an evidence browser second. The first screen must answer:

1. What should I do now?
2. Why?
3. What proof supports it?
4. Can I trust this run?

Raw signals remain available, but they must not be the primary homepage experience.
